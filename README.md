# COVID-19 Early Case Trend Analysis & Recovery Insights

An interactive analytics dashboard built for HealthGuard Analytics Pvt. Ltd. to analyze early-stage COVID-19 case data — patient demographics, infection spread patterns, recovery timelines, and regional impact — with a linear regression model to identify factors influencing recovery duration.

**Live app:** https://covidanalysis10.streamlit.app/
## Features
- Demographic breakdown by age and gender
- Infection spread patterns (reasons, order, contact exposure)
- Recovery duration trends over time and by region
- Confirmed vs. released case comparison across regions
- Linear regression model predicting recovery duration, with significance testing

## Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Statsmodels, Streamlit.
